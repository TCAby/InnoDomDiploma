In front of you is a system for creating and conducting online surveys.

Features:

  1. Unlimited number of surveys.
    1. Each survey can have an activity status and is additionally controlled by the time frame of the of the survey.
    2. Each survey can be available to everyone or only to registered users.
    3. For each survey, you can choose whether it is mandatory to answer each question, or if the respondent can skip some questions
  2. Each survey can have an unlimited number of questions.
    1. Each question can have an unlimited number of answers
    2. Each question can have only one correct answer or multiple answers. In this case, the question is considered answered correctly either when all correct answers are selected or when one (any) of the correct answers is selected.
In addition to the features available on the system's website, there is an API that allows you to run (conduct) surveys on your own website.
